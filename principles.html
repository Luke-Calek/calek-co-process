Overview

The principles outlined in this document trancend the different work processes (listed below) and are the core tasks that a UX designer does to create an experience.

Principles:

  • Understand
  • Ideate
  • Test
  • Implement

These steps are often presented in a linear left-to-right journey, but for a sucsessful UX project use principles should be repeated as part of a cycle of creation. Once the solution is created, smaller cycles should be continually deployed to maintain and evolve the product or service to meet changing user needs.
