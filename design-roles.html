XD, UX, UI - What is the difference?

While there are many interpretations of these different accronyms, the one thing in common is that there is always a lot of overlap between each field.

My definition of these fields is as follows:

  • Experience Design (XD) encompases the whole field of digital design.
  • User Experience (UX) is primarily focused on thinking about what an experince should be.
  • User Interface (UI) Design is primarily focused on making an experience based on UX thinking.

This website and its links use these acryonyms interchangable at different points, or specificly in different situations. Don't get hung-up which acroyonym is best, focus on the task and the ideal outcome.
