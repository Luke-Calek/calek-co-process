Picking the Correct Starting Point

The UX process can start on any of the 4 principles, but the results will be improved by starting on the earliest possible section.

Starting from scratch?
Understanding the core problem helps align the team and create a strong base for future ideation. Visit the Understand section
Have a user need already?
Doing a wide range of ideation will help find the best idea that meets the user’s needs. Visit the Ideation section
Think you have a good idea?
Quickly test your idea/s with prototypes to see if they really work. Visit the Test section
Have a validated idea already?
Create the idea correctly with clear design and delivery documentation that fleshes out all the details ready for development handover. Visit the Implement section
Already have a launched product or service?
Thorough analysis and refinement can help improve a product and better align it to the user’s needs. Visit the Test section
